# 카플랫폼 스크립트 삽입 정리 (고객용)

고객 문의 시 참고할 수 있도록, 사이트에 삽입된 스크립트를 정리한 문서입니다.

---

## 1. Google Tag (gtag.js) – 광고·전환 추적

**위치:** `index.html` 상단 `<head>` 안

**역할:**
- Google Ads 전환·광고 추적
- 전환 ID: `AW-18005708937`, `AW-729683917`

**삽입 코드 예시:**
```html
<script async src="https://www.googletagmanager.com/gtag/js?id=AW-18005708937"></script>
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('js', new Date());
  gtag('config', 'AW-18005708937');
  gtag('config', 'AW-729683917');
</script>
```

**전환 이벤트:**  
문의 완료 시(상담 완료 모달 표시 시점) `script.js`에서 아래 전환 이벤트 전송  
- `send_to`: `AW-729683917/zo3DCL6ooZMbEM2v-NsC`  
- value: 1.0, currency: KRW

## 3. Google Apps Script – 문의 이메일·시트 연동

**위치:** `script.js` 내 상수

**역할:**
- 문의 신청 **완료** 시, 문의 내용을 Google Apps Script 웹앱으로 전송
- 해당 스크립트에서 **이메일 발송**(수신: andukgi@gmail.com) 및 **(선택) 구글 시트 기록**

**사용 URL (script.js 내):**
```javascript
const APPS_SCRIPT_WEB_APP_URL = 'https://script.google.com/macros/s/AKfycbw6eG32DIJ_yjjhNJTh9UQDP8nk1cF2QFrZZulq6WDLS9doMO9dfR6uB7IryhZwBZ9Y/exec';
```

**동작 흐름:**
1. 고객이 견적/문의 폼 제출
2. 기존대로 D1 DB에 문의 저장
3. 동시에 브라우저에서 위 URL로 문의 데이터 POST
4. Apps Script에서 메일 발송 + (설정 시) 시트에 한 줄 추가

**상세 설정·변경:** 프로젝트 내 `APPS_SCRIPT_SETUP.md` 참고

## 4. JSON-LD 구조화 데이터 (스키마)

**위치:** `index.html` `<head>` 안

**역할:**
- 검색엔진(Google 등)에 회사·서비스 정보 전달
- Organization, Service 스키마로 사이트 설명

**포함 내용 예:**
- 회사명: 카플랫폼 / 이지플랫폼
- 연락처: 1688-7446, ezplat@naver.com
- 주소: 경기도 고양시 일산동구 정발산로 19, 9층 901호
- 서비스 설명: AI 실시간 비교견적, 장기렌트/리스/할부

**형식:** `<script type="application/ld+json">` 안에 JSON으로 삽입

## 7. 콘솔에서 동작 확인 (개발자 도구 F12 → Console)

페이지 로드 시 및 문의 완료 시 아래 메시지가 나오면 해당 스크립트가 정상 동작하는 것입니다.

| 시점 | 콘솔 메시지 | 의미 |
|------|-------------|------|
| 페이지 로드 | `[카플랫폼] Google Tag (gtag.js) 설정됨 - AW-18005708937, AW-729683917` | Google Tag 로드됨 |
| 페이지 로드 | `[카플랫폼] 네이버 사이트 소유 확인 메타: 설정됨` | 네이버 인증 메타 있음 |
| 페이지 로드 | `[카플랫폼] JSON-LD 구조화 데이터(Organization, Service) 로드됨` | 스키마 데이터 로드됨 |
| script.js 로드 후 | `[카플랫폼] script.js 로드됨, Apps Script 연동 URL: 설정됨` | Apps Script URL 설정됨 |
| 페이지 로드 | `[카플랫폼] AceCounter 스크립트 삽입됨 (AZ2A105497)` | AceCounter 삽입됨 |
| 문의 제출 시 | `[카플랫폼] Apps Script로 문의 데이터 전송 시도` | 이메일/시트 연동 요청 시작 |
| 문의 제출 시 | `[카플랫폼] Apps Script 전송 요청 완료 (no-cors)` | Apps Script 전송 요청 완료 |
| 문의 완료 모달 시 | `[카플랫폼] Google 전환 이벤트 전송됨 (AW-729683917/...)` | 전환 이벤트 전송됨 |

**gtag가 안 불러와진 경우:**  
`[카플랫폼] gtag 미로드 - 전환 이벤트 미전송` 이 나오면 Google Tag 스크립트가 막혔거나 늦게 로드된 상태입니다.

## 요약 (고객에게 한 줄로 설명할 때)

- **Google Tag:** Google 광고·전환 측정용 스크립트 (전환 시 전송)
- **AceCounter:** 방문자 통계 수집용 스크립트
- **Google Apps Script:** 문의 완료 시 이메일 발송·시트 기록용 연동
- **JSON-LD:** 검색엔진용 회사/서비스 정보 구조화 데이터
- **네이버 메타:** 네이버 사이트 소유 인증용 메타 태그

문의 시 이 문서를 공유하시면 됩니다.
